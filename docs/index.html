📖 使用指南

博客主题

本博客使用 NexT 主题，这是一个高质量、优雅的 Hexo 主题。

主要功能

  • 响应式设计：完美适配各种设备
  • 深色主题：护眼的深色配色方案
  • 搜索功能：快速查找文章内容
  • 目录导航：自动生成文章目录
  • 社交分享：便捷的社交媒体分享

🛠 技术栈

  • 静态站点生成器：Hexo
  • 主题：NexT 8.x
  • 部署平台：GitHub Pages
  • 样式预处理器：Stylus
  • 图标：Font Awesome
