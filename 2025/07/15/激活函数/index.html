激活函数

Linear Activation

f(x)=xf(x) = x f(x)=x

ReLU

f(x)=max⁡(0, x)f(x) = \max(0,\, x) f(x)=max(0,x)

隐藏层中最常用，计算高效，缓解梯度消失

Sigmoid

f(x)=11+e−xf(x) = \frac{1}{1+\mathrm{e}^{-x}} f(x)=1+e−x1​

二分类输出层，输出概率值

Tanh

f(x)=tanh⁡(x)=ex−e−xex+e−xf(x) = \tanh(x) = \frac{\mathrm{e}^{x}-\mathrm{e}^{-x}}{\mathrm{e}^{x}+\mathrm{e}^{-x}} f(x)=tanh(x)=ex+e−xex−e−x​

RNN 隐藏层，输出范围 [-1, 1]，零中心化

Leaky ReLU

深层网络中替代 ReLU，提升训练稳定性

Softplus

f(x)=ln⁡ ⁣(1+ex)f(x) = \ln\!\bigl(1+\mathrm{e}^{x}\bigr) f(x)=ln(1+ex)

ReLU 的平滑近似，用于需要平滑激活的场景，如概率模型

ELU

f(x)={x,x≥0α (ex−1),xlt;0f(x)= \begin{cases} x, & x\ge 0\\[4pt] \alpha\,(\mathrm{e}^{x}-1), & x&lt;0 \end{cases} f(x)={x,α(ex−1),​x≥0x​lt;0​

加速收敛，输出均值接近 0，适合深层网络

f(x)=λ{x,x≥0α (ex−1),xlt;0f(x)=\lambda \begin{cases} x, & x\ge 0\\[4pt] \alpha\,(\mathrm{e}^{x}-1), & x&lt;0 \end{cases} f(x)=λ{x,α(ex−1),​x≥0x​lt;0​

类似 ELU，但斜率更大，自归一化网络中，保持层间分布稳定

深度模型中性能优于 ReLU，如 EfficientNet

GELU

f(x)=x⋅Φ(x)=x2[1+erf ⁣(x2)]f(x)=x\cdot\Phi(x)=\frac{x}{2}\left[1+\mathrm{erf}\!\left(\frac{x}{\sqrt{2}}\right)\right] f(x)=x⋅Φ(x)=2x​[1+erf(2​x​)]

类似 Swish，但更平滑，Transformer、BERT、GPT 等模型中广泛使用

激活函数 非线性 输出范围 常用于 特点
Linear 否 (-∞, ∞) 回归输出 简单
ReLU 是 [0, ∞) 隐藏层 快速、稀疏
Sigmoid 是 (0, 1) 二分类输出 易饱和
Tanh 是 (-1, 1) RNN 零中心化
LeakyReLU 是 (-∞, ∞) 隐藏层 解决死神经元
Softplus 是 (0, ∞) 平滑 ReLU 可导
ELU 是 (-α, ∞) 深层网络 加速收敛
SELU 是 类似 ELU 自归一化网络 自动归一化
Swish 是 (-∞, ∞) 深层模型 非单调
GELU 是 (-∞, ∞) Transformer 高斯加权
